# Online-Exam-Platform

This project is an implementation of a simplified online examination platform.

## Description

The application simulates a system similar to Moodle, capable of managing exams, questions, students, and evaluations.

It processes commands from input files and executes them in chronological order, generating output files with the results. :contentReference[oaicite:0]{index=0}

## Features

- Exam creation and management
- Multiple types of questions:
  - Open-ended
  - Fill in the blank
  - Multiple choice
- Student registration
- Answer submission and automatic grading
- Score tracking per student
- Report generation for exams

## Functionality

- Add questions to exams with difficulty and score
- Sort and display questions based on different criteria
- Submit exams and compute scores
- Handle submissions outside the allowed time interval
- Generate reports with student results

## Technologies

- Java
- OOP principles (inheritance, polymorphism, abstraction)
- Collections and generics
- File I/O

## How to run

Run the main class with the test folder name as argument. The program reads commands from input files and writes results to output files.

## Notes

The project follows the assignment requirements and focuses on clean object-oriented design and correct use of collections and generics.
